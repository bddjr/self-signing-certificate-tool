## Build from source

Need dependencies:
> [Git](https://git-scm.com)  
> [Go](https://go.dev)  
> [Node.js](https://nodejs.org)  
> [Pnpm](https://pnpm.io)  

If on Windows, use git bash.

```
git clone https://github.com/bddjr/self-signing-certificate-tool
cd self-signing-certificate-tool
pnpm i
chmod +x build.sh
./build.sh
```
